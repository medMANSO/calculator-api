# calculator-api

1. ".env"
    - Create a ".env" file referring to ".env.example"

2. Runing the project locally
    - npm install
    - npm run devstart
  
3. Running the project in a Docker container
    - ./_buildProject.sh
    - ./_start.sh
    - Stopping the container
      - ./_stop.sh
 
4. Running Tests
    - npm install
    - npm run test
